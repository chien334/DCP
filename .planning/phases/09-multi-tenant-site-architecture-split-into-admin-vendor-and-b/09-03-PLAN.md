## 1. Create Buyer-specific Layout

Create directory `src/ProcureFlow.Buyer/Components/Pages/Buyer/`:
- `Dashboard.razor` - Procurement KPIs, pending approvals, recent activity
- `RfpList.razor` - All RFPs with status filters
- `RfpCreate.razor` - Multi-step RFP creation wizard
- `RfpDetail.razor` - RFP detail with item/spec management
- `VendorInvite.razor` - Invite vendors to RFP
- `BidComparison.razor` - Side-by-side bid comparison matrix
- `Finalize.razor` - Winning bid selection and finalization
- `ContractList.razor` - All contracts with status
- `ContractView.razor` - Contract detail and signature tracking

## 3. Create Buyer Shared Components

Create `src/ProcureFlow.Buyer/Components/Shared/`:
- `BuyerNavMenu.razor` - Full procurement workflow navigation
- `RfpCard.razor` - RFP status card with progress indicator
- `BidComparisonTable.razor` - Side-by-side comparison grid
- `WorkflowStepper.razor` - Multi-step workflow progress indicator
- `ApprovalBadge.razor` - Status badges (Draft, Pending, Approved, etc.)
- `ItemTable.razor` - Line item display with specs

## 4. Create Buyer Program.cs

Create `src/ProcureFlow.Buyer/Program.cs`:
- Blazor Web App with SSR
