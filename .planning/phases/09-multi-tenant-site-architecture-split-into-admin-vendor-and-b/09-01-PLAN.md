# Admin Portal Architecture

## 1. Create Admin-specific Layout

Create `src/ProcureFlow.Admin/Components/Layout/AdminLayout.razor`:
- Professional sidebar navigation with icons
- Top header with company logo area and user menu
- Main content area with breadcrumb navigation
- Collapsible sidebar for dashboard space
- Color scheme: Deep blue primary (#1E3A5F), slate gray accents (#64748B), white content areas
- Typography: Inter font family, clean hierarchy

## 2. Create Admin Pages Structure

Create directory `src/ProcureFlow.Admin/Components/Pages/Admin/`:
- `Dashboard.razor` - System overview with KPIs
- `Companies.razor` - Company CRUD with data table
- `Employees.razor` - Employee management with role assignment
- `Settings.razor` - System configuration
- `AuditLog.razor` - Audit trail viewer

## 3. Create Shared Admin Components

Create `src/ProcureFlow.Admin/Components/Shared/`:
- `AdminNavMenu.razor` - Navigation sidebar with icons
- `DataTable.razor` - Reusable data table with sorting/pagination
- `StatCard.razor` - Dashboard KPI cards
- `FormField.razor` - Standardized form inputs

## 4. Create Admin Program.cs

Create `src/ProcureFlow.Admin/Program.cs`:
- Blazor Web App with SSR
- Register Admin API endpoints
- Configure admin-specific authentication policies
- Set up dependency injection for admin services

## 5. Create Admin App.razor and Routes

Create routing structure for Admin portal with `/admin/*` paths.
